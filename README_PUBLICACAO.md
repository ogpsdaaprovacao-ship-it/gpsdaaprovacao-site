# Publicação GPS Studio

Projeto: PMESP
URL prevista: https://gpsdaaprovacao.com/pmesp/

## Como publicar pelo caminho mais simples

### Opção recomendada: Cloudflare Pages + GitHub
1. Crie ou abra o repositório das suas landing pages.
2. Descompacte este ZIP.
3. Se o modo escolhido foi pasta no domínio, envie a pasta `pmesp/` inteira para a raiz do repositório.
4. Se o modo escolhido foi subdomínio, envie os arquivos da pasta exportada para o repositório/projeto específico daquele subdomínio.
5. No Cloudflare Pages, conecte o repositório.
6. Configure o domínio em Custom Domains.
7. Teste a URL final.

### GitHub Pages puro
1. Envie os arquivos para o repositório.
2. Ative Settings > Pages.
3. Para domínio próprio, configure o domínio nas opções do GitHub Pages e aponte o DNS.

## Observação
Não apague a pasta assets. Ela contém imagens e vídeos usados pela página.
